== Installation ==

= Automatic installation =

Automatic installation is the easiest option. Log in to your WordPress dashboard, navigate to the Plugins menu, and click “Add New.”
 
In the search field type “SEO X,” then click “Search Plugins.” Once you’ve found us,  you can view details about it such as the point release, rating, and description. You can install it by clicking the “Install Now” button,  WordPress will install it immediately to your website.

= Manual installation =

1. Upload `seox` zip file to the `/wp-content/plugins/` directory. 
2. Activate SEO X through the `Plugins` menu in WordPress.
3. Done !!!
